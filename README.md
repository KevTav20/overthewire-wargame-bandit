# overthewire-wargame-bandit

Este repositorio alberga las soluciones que utilicé para resolver cada nivel del wargame Bandit de OverTheWire, junto con el procedimiento que seguí en cada uno: desde probar los comandos disponibles, hasta investigar sus modificadores y ver si era necesario combinarlos.

## Estructura

Cada nivel tiene su propia carpeta, con un archivo `solucion.md` que incluye:

- El proceso completo que seguí para encontrar la contraseña del siguiente nivel
- Capturas de pantalla como evidencia de que la contraseña fue encontrada
- El historial de comandos utilizado

Las contraseñas aparecen parcialmente censuradas en el historial, para no arruinarle la solución a quien todavía esté resolviendo el reto.
